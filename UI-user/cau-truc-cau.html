Cấu trúc câu thông dụng

Các cấu trúc tiếng Anh thường gặp trong giao tiếp và công việc.

Too / Enough
too + adj/adv + to V
I’m too tired to work.
So / Such
so + adj / such + a/an + noun
It’s so hot today.
Would rather
would rather + V
I’d rather stay home.
Used to
used to + V
I used to smoke.
Because / So
because + clause
I stayed home because it rained.
In order to
in order to + V
I study English to improve my job.
